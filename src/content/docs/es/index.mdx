## Aspectos destacados de la arquitectura

- **Flows** - Grafos de orquestación definidos en YAML con nodos de componentes WASM
- **Packs** - Archivos `.gtpack` firmados que contienen flows, componentes y assets
- **Components** - Bloques de construcción WASM portables que implementan la interfaz WIT de Greentic
- **Providers** - Componentes WASM que conectan servicios externos (mensajería, eventos, secretos)
