## Requisitos del sistema

### Requisitos mínimos

| Componente | Requisito |
|-----------|-------------|
| Sistema operativo | Linux, macOS, Windows |
| Rust | 1.95+ |
| Memoria | 4 GB RAM |
| Disco | 2 GB de espacio libre |

### Recomendado

| Componente | Requisito |
|-----------|-------------|
| Rust | 1.95+ (latest supported) |
| Memoria | 8 GB RAM |
| CPU | Procesador multinúcleo |

## Instalar Rust

Si no tienes Rust instalado:

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Establece la versión correcta del toolchain:

```bash
rustup default 1.95
```

Agrega el target WASM para construir componentes:

```bash
rustup target add wasm32-wasip2
```

## Instalar GTC CLI

Instala primero el router `gtc` y deja que `gtc install` instale las herramientas complementarias de Greentic. No instales `greentic-cli` ni crates complementarios individuales para uso normal; `gtc install` mantiene los binarios delegados alineados con el router.

<Tabs>
<TabItem label="cargo-binstall (Recomendado)">

```bash
cargo install cargo-binstall
cargo binstall gtc
gtc install
gtc doctor
```

</TabItem>
<TabItem label="Desde el código fuente">

```bash
git clone https://github.com/greenticai/greentic.git
cd greentic/greentic
cargo build --release

# Add to PATH
export PATH="$PATH:$(pwd)/target/release"
gtc install
gtc doctor
```

</TabItem>
<TabItem label="Binarios precompilados">

Descarga desde [GitHub Releases](https://github.com/greenticai/greentic/releases):

```bash
# macOS (Apple Silicon)
curl -LO https://github.com/greenticai/greentic/releases/latest/download/gtc-darwin-arm64.tar.gz
tar -xzf gtc-darwin-arm64.tar.gz
sudo mv gtc /usr/local/bin/

# macOS (Intel)
curl -LO https://github.com/greenticai/greentic/releases/latest/download/gtc-darwin-x64.tar.gz

# Linux (x86_64)
curl -LO https://github.com/greenticai/greentic/releases/latest/download/gtc-linux-x64.tar.gz

# Windows
# Download gtc-windows-x64.zip from releases
```

</TabItem>
</Tabs>

Para providers basados en webhooks (Telegram, Slack, etc.):

<Tabs>

```bash
brew install ngrok/ngrok/ngrok  # macOS
```

<TabItem label="Cloudflared">

```bash
# Run
cloudflared tunnel --url http://localhost:8080
```

</TabItem>
</Tabs>

## Herramientas de desarrollo

### Toolchain de Greentic

Instala o actualiza los binarios Greentic delegados que usa el router `gtc`:

```bash
gtc install
```

Usa `gtc install --dry-run` para previsualizar el plan de instalación. Para entornos controlados, fija la toolchain:

```bash
gtc install --release 1.0.15
gtc install --manifest ./greentic-toolchain.json
```

### Creación de componentes

`gtc install` instala las herramientas complementarias estándar. Para desarrollo independiente de componentes también puedes instalar `cargo-component`:

```bash
cargo install --locked cargo-component
```

Luego usa la superficie delegada de componentes:

```bash
gtc dev component --help
```

## Verificar tu entorno

Ejecuta lo siguiente para verificar que todo funciona:

```bash
# Check GTC CLI
gtc --help

# Check Rust WASM target
rustup target list --installed | grep wasm32-wasip2

# Create and run a test bundle
gtc wizard --dry-run
```

## Resolución de problemas

### Problemas comunes

**Error: "wasm32-wasip2 target not found"**
```bash
rustup target add wasm32-wasip2
```

**Error: "Permission denied"**
```bash
# On Linux/macOS, ensure binary is executable
chmod +x /usr/local/bin/gtc
```
