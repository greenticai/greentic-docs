## システム要件

### 最小要件

| Component | 要件 |
|-----------|------|
| OS | Linux, macOS, Windows |
| Rust | 1.95+ |
| Memory | 4 GB RAM |
| Disk | 2 GB の空き容量 |

### 推奨

| Component | 要件 |
|-----------|------|
| Rust | 1.95+ (latest supported) |
| Memory | 8 GB RAM |
| CPU | マルチコア processor |

## Rust のインストール

Rust がインストールされていない場合:

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

正しい toolchain version を設定します:

```bash
rustup default 1.95
```

component を build するための WASM target を追加します:

```bash
rustup target add wasm32-wasip2
```

## GTC CLI のインストール

まず `gtc` router をインストールし、その後 `gtc install` で Greentic companion tools をインストールします。通常利用では `greentic-cli` や個別の companion crates を直接インストールしないでください。`gtc install` が delegated binaries を router と揃えます。

<Tabs>
<TabItem label="cargo-binstall (Recommended)">

```bash
cargo install cargo-binstall
cargo binstall gtc
gtc install
gtc doctor
```

</TabItem>
<TabItem label="From Source">

```bash
git clone https://github.com/greenticai/greentic.git
cd greentic/greentic
cargo build --release

# Add to PATH
export PATH="$PATH:$(pwd)/target/release"
gtc install
gtc doctor
```

</TabItem>
<TabItem label="Pre-built Binaries">

[GitHub Releases](https://github.com/greenticai/greentic/releases) からダウンロードします:

```bash
# macOS (Apple Silicon)
curl -LO https://github.com/greenticai/greentic/releases/latest/download/gtc-darwin-arm64.tar.gz
tar -xzf gtc-darwin-arm64.tar.gz
sudo mv gtc /usr/local/bin/

# macOS (Intel)
curl -LO https://github.com/greenticai/greentic/releases/latest/download/gtc-darwin-x64.tar.gz

# Linux (x86_64)
curl -LO https://github.com/greenticai/greentic/releases/latest/download/gtc-linux-x64.tar.gz

# Windows
# Download gtc-windows-x64.zip from releases
```

</TabItem>
</Tabs>

## Development Tools

### Greentic Toolchain

`gtc` router が使う delegated Greentic binaries をインストールまたは更新します:

```bash
gtc install
```

Install plan を確認するには `gtc install --dry-run` を使います。Controlled environments では toolchain を pin できます:

```bash
gtc install --release 1.0.15
gtc install --manifest ./greentic-toolchain.json
```

### Component Authoring

`gtc install` は standard companion tools をインストールします。Standalone component development では `cargo-component` も利用できます:

```bash
cargo binstall cargo-component
```

その後、delegated component surface を使います:

```bash
gtc dev component --help
```

## セットアップの検証

すべてが正しく動作していることを確認するには、次を実行します:

```bash
# Check GTC CLI
gtc --help

# Check Rust WASM target
rustup target list --installed | grep wasm32-wasip2

# Create and run a test bundle
gtc wizard --dry-run
```

## トラブルシューティング

### よくある問題

**Error: "wasm32-wasip2 target not found"**
```bash
rustup target add wasm32-wasip2
```

**Error: "Permission denied"**
```bash
# On Linux/macOS, ensure binary is executable
chmod +x /usr/local/bin/gtc
```
