## 主な特徴

<CardGrid stagger>
	<Card title="WASM Components" icon="puzzle">
		WebAssembly（WASI Preview 2）による portable かつ sandboxed execution。1 度ビルドすれば、どこでも実行できます。
	</Card>
	<Card title="Multi-Tenant" icon="setting">
		安全でスケーラブルな deployment のために、`TenantCtx` による tenant isolation を標準で備えています。
	</Card>
	<Card title="Flow-Based Orchestration" icon="random">
		YAML ベースの flow graph（`.ygtc` files）で automation pipeline を定義します。
	</Card>
	<Card title="Provider Ecosystem" icon="external">
		WASM provider packs を通じて Slack、Teams、Telegram、WhatsApp などに接続できます。
	</Card>

## アーキテクチャのハイライト

- **Flows** - WASM component node のオーケストレーション graph を YAML で定義
- **Packs** - flows、components、assets を含む署名付き `.gtpack` archive
- **Components** - Greentic WIT interface を実装する portable な WASM building block
- **Providers** - 外部 service（messaging、events、secrets）を橋渡しする WASM component
