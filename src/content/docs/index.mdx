## Architecture Highlights

- **Flows** - YAML-defined orchestration graphs of WASM component nodes
- **Packs** - Signed `.gtpack` archives containing flows, components, and assets
- **Components** - Portable WASM building blocks implementing the Greentic WIT interface
- **Providers** - WASM components bridging external services (messaging, events, secrets)
