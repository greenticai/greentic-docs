## Sorotan Arsitektur

- **Flow** - Graf orkestrasi berbasis YAML dari node komponen WASM
- **Pack** - Arsip `.gtpack` bertanda tangan berisi flow, komponen, dan aset
- **Komponen** - Building block WASM portabel yang mengimplementasi antarmuka WIT Greentic
- **Provider** - Komponen WASM yang menjembatani layanan eksternal (messaging, event, secret)
