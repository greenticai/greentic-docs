## Persyaratan Sistem

### Persyaratan Minimum

| Komponen | Persyaratan |
|----------|-------------|
| OS | Linux, macOS, Windows |
| Rust | 1.95+ |
| Memori | 4 GB RAM |
| Disk | 2 GB ruang kosong |

### Rekomendasi

| Komponen | Persyaratan |
|----------|-------------|
| Rust | 1.95+ (terbaru yang didukung) |
| Memori | 8 GB RAM |
| CPU | Prosesor multi-core |

## Menginstal Rust

Jika Anda belum menginstal Rust:

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Atur versi toolchain yang benar:

```bash
rustup default 1.95
```

Tambahkan target WASM untuk build komponen:

```bash
rustup target add wasm32-wasip2
```

## Menginstal GTC CLI

Instal router `gtc` terlebih dahulu, lalu biarkan `gtc install` menginstal tool pendamping Greentic. Jangan instal `greentic-cli` atau crate pendamping satu per satu untuk penggunaan normal; `gtc install` menjaga binary yang didelegasikan tetap selaras dengan router.

<Tabs>
<TabItem label="cargo-binstall (Rekomendasi)">

```bash
cargo install cargo-binstall
cargo binstall gtc
gtc install
gtc doctor
```

</TabItem>
<TabItem label="Dari Source">

```bash
git clone https://github.com/greenticai/greentic.git
cd greentic/greentic
cargo build --release

# Tambahkan ke PATH
export PATH="$PATH:$(pwd)/target/release"
gtc install
gtc doctor
```

</TabItem>
<TabItem label="Binary Pre-built">

Download dari [GitHub Releases](https://github.com/greenticai/greentic/releases):

```bash
# macOS (Apple Silicon)
curl -LO https://github.com/greenticai/greentic/releases/latest/download/gtc-darwin-arm64.tar.gz
tar -xzf gtc-darwin-arm64.tar.gz
sudo mv gtc /usr/local/bin/

# macOS (Intel)
curl -LO https://github.com/greenticai/greentic/releases/latest/download/gtc-darwin-x64.tar.gz

# Linux (x86_64)
curl -LO https://github.com/greenticai/greentic/releases/latest/download/gtc-linux-x64.tar.gz

# Windows
# Download gtc-windows-x64.zip dari releases
```

</TabItem>
</Tabs>

## Tool Development

### Greentic Toolchain

Instal atau perbarui binary Greentic yang didelegasikan dan digunakan oleh router `gtc`:

```bash
gtc install
```

Gunakan `gtc install --dry-run` untuk melihat rencana instalasi. Untuk lingkungan terkontrol, pin toolchain:

```bash
gtc install --release 1.0.15
gtc install --manifest ./greentic-toolchain.json
```

### Pembuatan Komponen

`gtc install` menginstal tool pendamping standar. Untuk pengembangan komponen mandiri, Anda juga dapat menginstal `cargo-component`:

```bash
cargo install --locked cargo-component
```

Lalu gunakan permukaan komponen yang didelegasikan:

```bash
gtc dev component --help
```

## Memverifikasi Setup Anda

Jalankan perintah berikut untuk memverifikasi semuanya berfungsi:

```bash
# Cek GTC CLI
gtc --help

# Cek target Rust WASM
rustup target list --installed | grep wasm32-wasip2

# Buat dan jalankan bundle tes
gtc wizard --dry-run
```

## Pemecahan Masalah

### Masalah Umum

**Error: "wasm32-wasip2 target not found"**
```bash
rustup target add wasm32-wasip2
```

**Error: "Cannot connect to NATS"**
```bash
# Jalankan NATS embedded dengan gtc start
gtc start ./my-bundle --nats on
```

**Error: "Permission denied"**
```bash
# Di Linux/macOS, pastikan binary bisa dieksekusi
chmod +x /usr/local/bin/gtc
```
