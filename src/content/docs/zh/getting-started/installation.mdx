## 系统要求

### 最低要求

| 组件 | 要求 |
|-----------|-------------|
| OS | Linux、macOS、Windows |
| Rust | 1.95+ |
| Memory | 4 GB RAM |
| Disk | 2 GB 可用空间 |

### 推荐配置

| 组件 | 要求 |
|-----------|-------------|
| Rust | 1.95+（最新支持版本） |
| Memory | 8 GB RAM |
| CPU | 多核处理器 |

## 安装 Rust

如果你还没有安装 Rust：

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

设置正确的 toolchain 版本：

```bash
rustup default 1.95
```

添加用于构建 components 的 WASM target：

```bash
rustup target add wasm32-wasip2
```

## 安装 GTC CLI

先安装 `gtc` router，然后让 `gtc install` 安装 Greentic companion tools。正常使用时不要直接安装 `greentic-cli` 或单独的 companion crates；`gtc install` 会让 delegated binaries 与 router 保持一致。

<Tabs>
<TabItem label="cargo-binstall (Recommended)">

```bash
cargo install cargo-binstall
cargo binstall gtc
gtc install
gtc doctor
```

</TabItem>
<TabItem label="From Source">

```bash
git clone https://github.com/greenticai/greentic.git
cd greentic/greentic
cargo build --release

# Add to PATH
export PATH="$PATH:$(pwd)/target/release"
gtc install
gtc doctor
```

</TabItem>
<TabItem label="Pre-built Binaries">

从 [GitHub Releases](https://github.com/greenticai/greentic/releases) 下载：

```bash
# macOS (Apple Silicon)
curl -LO https://github.com/greenticai/greentic/releases/latest/download/gtc-darwin-arm64.tar.gz
tar -xzf gtc-darwin-arm64.tar.gz
sudo mv gtc /usr/local/bin/

# macOS (Intel)
curl -LO https://github.com/greenticai/greentic/releases/latest/download/gtc-darwin-x64.tar.gz

# Linux (x86_64)
curl -LO https://github.com/greenticai/greentic/releases/latest/download/gtc-linux-x64.tar.gz

# Windows
# Download gtc-windows-x64.zip from releases
```

</TabItem>
</Tabs>

用于生产环境中的 session 持久化：

<Tabs>


```bash

redis-server
```

</TabItem>
</Tabs>

### Ngrok/Cloudflared（公共 URL）

用于基于 webhook 的 providers（Telegram、Slack 等）：

<Tabs>

```bash

brew install ngrok/ngrok/ngrok  # macOS
# or download from https://ngrok.com/download
```

</TabItem>
<TabItem label="Cloudflared">

```bash
# Run
cloudflared tunnel --url http://localhost:8080
```

</TabItem>
</Tabs>

## 开发工具

### Greentic Toolchain

安装或更新 `gtc` router 使用的 delegated Greentic binaries：

```bash
gtc install
```

使用 `gtc install --dry-run` 预览安装计划。对于受控环境，可以固定 toolchain：

```bash
gtc install --release 1.0.15
gtc install --manifest ./greentic-toolchain.json
```

### Component Authoring

`gtc install` 会安装标准 companion tools。对于独立 component development，也可以安装 `cargo-component`：

```bash
cargo binstall cargo-component
```

然后使用 delegated component surface：

```bash
gtc dev component --help
```

## 验证你的环境

运行以下命令以确认一切正常：

```bash
# Check GTC CLI
gtc --help

# Check Rust WASM target
rustup target list --installed | grep wasm32-wasip2

# Create and run a test bundle
gtc wizard --dry-run
```

## 故障排查

### 常见问题

**错误："wasm32-wasip2 target not found"**
```bash
rustup target add wasm32-wasip2
```

**错误："Permission denied"**
```bash
# On Linux/macOS, ensure binary is executable
chmod +x /usr/local/bin/gtc
```
