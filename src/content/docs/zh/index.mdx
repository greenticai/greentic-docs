## 架构亮点

- **Flows** - 使用 YAML 定义、由 WASM component 节点组成的编排图
- **Packs** - 带签名的 `.gtpack` 归档，包含 flows、components 和 assets
- **Components** - 实现 Greentic WIT 接口的可移植 WASM 构建块
- **Providers** - 连接外部服务的 WASM components（消息、事件、secrets）
