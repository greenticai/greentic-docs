# 入门模板

**入门模板**是用户在 Designer 中创建新项目时可以选择的、经过审核的
起点。用户无需从空白画布开始，而是从一个
已经体现你所在组织约定的流程起步。

操作员策划租户——或特定团队——能看到哪些模板，
因此用户打开的画廊始终与他们相关。

## 策划模板（操作员）

在管理控制台中，平台操作员选择为某个租户
呈现哪些模板。策划是按用户解析的：用户看到的画廊是
租户范围模板与为该用户所属团队
策划的模板的并集。

<Aside type="note">
由于解析后的集合取决于用户的团队成员身份，
同一租户中的两个用户可能看到不同的入门画廊——每人看到的恰好是
为其团队策划的模板加上租户范围的模板。
</Aside>

## 从模板开始（用户）

在 Designer 中，用户打开 **New project** 对话框并选择 **Start
from a template**。选择器会列出策划好的入门模板；选中其中一个
会用该模板的流程预填充新项目，随即可供编辑。

选择空白选项仍然像以前一样有效——模板是
可选的领先一步，绝非必需。

## 什么样的入门模板才算好

- 名称和摘要清晰、具有描述性，让用户可以挑选出合适的那一个
  而无需打开它。
- 流程完整到足以运行，但又足够通用以便适配。
- 将你的团队应遵循的约定——命名、节点结构、渠道接线
  ——内置其中，使模板既能教学也能加速。
